ScrollSpy

Scrollspy é um plugin jQuery que monitora certos elementos e qual elemento está centralizado na tela do usuário. Nossa principal demonstração disso é a nossa tabela de conteúdo em todas as páginas de documentação à direita. Ao clicar nesses links também irá rolar a página para o elemento.
