Tabs - Introdução

A estrutura de abas consiste um uma lista de abas com identificadores correspondentes ao id da aba. Quando você clicar em uma aba, apenas o elemento com o id correspondente ao identificador da aba será visível. Você pode adicionar a classe .disabled para fazer a aba inacessível
